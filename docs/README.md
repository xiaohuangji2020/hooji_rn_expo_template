## 快速开始

### 检查代码规范

```bash
npm run lint
```

### 自动修复问题

```bash
npm run lint:fix
```

### VS Code 自动修复

项目已配置 `.vscode/settings.json`，安装 ESLint 扩展后，保存文件会自动修复大部分问题。
